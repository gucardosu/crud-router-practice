# CRUD de Contatos

Um projeto simples de CRUD (Create, Read, Update, Delete) de contatos, desenvolvido com Node.js com o objetivo de praticar conceitos de roteamento (router) e organização de aplicações no backend.

## Funcionalidades

- Criar um novo contato  
- Listar todos os contatos  
- Buscar um contato por ID  
- Atualizar informações de um contato  
- Remover um contato  
- Organização de rotas para cada operação  

## Tecnologias utilizadas

- Node.js  
- (adicione aqui se usou algo mais, por exemplo:)
  - Express  
  - Nodemon
